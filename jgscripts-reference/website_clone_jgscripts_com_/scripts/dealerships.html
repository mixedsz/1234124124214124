🎨 Fully Rebuilt with Modern UI

Version 2 has been rebuilt from the ground up with a refined, modern interface and new backend architecture, providing a cleaner player experience, smoother staff workflows, and a scalable foundation designed for long-term server growth rather than short-term feature patches.

🏪 Public & Player-Owned Dealerships

JG Dealerships supports both self-service public showrooms and fully player-owned businesses, allowing servers to run standard vehicle shops alongside realistic dealership enterprises where owners manage stock, employees, sales, financing, and promotions, with each location able to operate under different rules and structures.

📍 In-Game Dealership Creator

Admins can create, edit, and manage dealerships entirely in-game using the redesigned admin panel, placing interaction points, defining zones, and adjusting location settings on the fly. Setting up a dealership script has never been this easy.

🧠 Advanced Interaction System

The system supports target integrations, radial menus, DUI text prompts, text UI and markers, with interactions attachable to peds, objects, vehicles, polyzones, or simple radiuses, allowing every dealership location to be set up exactly the way you want it.

🚗 Showroom Display Vehicles

Display vehicles can be placed, moved, and updated directly in-game by dealership staff, allowing showrooms to reflect real stock, highlight featured vehicles, and create visually immersive dealership floors to entice customers to purchase.

📱 Sell or Test Drive in RP

Dealership employees can handle vehicle purchases directly with customers through an in-game tablet while inside their dealership, managing financing, colour selection, and test drives as part of RP rather than players completing everything themselves through a static showroom interface.

🚚 Stock Delivery Trucking Missions

Vehicle ordering can optionally become gameplay through trucking missions where ordered stock is collected using a car trailer and the ordered vehicles physically appear on the trailer. Multiple orders can be delivered per run depending on capacity, while servers that prefer a simpler setup can use automated timed deliveries instead.

🎨 Advanced Vehicle Colour Control

Servers can configure colour selection to allow any RGB colour via a colour wheel, restrict purchases to a curated RGB palette, or use a limited list of colour IDs, including support for special finishes such as chameleon colours.

💸 Advanced Financing Included

A powerful and configurable financing system allows dealerships to offer payment plans with interest, enabling more realistic vehicle ownership and expanding RP opportunities around debt, repayments, and business income.

🎟️ Coupons & Promotions

Built-in coupon code support allows servers and dealership owners to run discounts, seasonal sales, and promotional events to drive activity and create dynamic dealership economies.

🪙 Custom Currency API

The custom currency API allows dealerships to accept VIP coins, tokens, or alternative economy systems, making it easy to integrate with existing server monetisation structures.

📊 Sales Dashboards for Staff

Dealership employees receive a redesigned management dashboard featuring detailed sales reporting, helping dealership businesses monitor performance and operate with real business-style oversight.

⚡ Major Performance Improvements

The rewritten backend includes significant efficiency optimisations that reduce resource usage while supporting more advanced systems and interaction types than the previous version.
